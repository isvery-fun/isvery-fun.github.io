how to make an alias?

if you want your isvery.fun subdomain to be an alias (when you visit site through the subdomain, it shows the page content instead of redirecting to your IP or domain), you need to follow these steps (instructions for cloudflare):

1. go to the DNS section of the domain management

2. add a new DNS record of the CNAME type:

- in the name field, specify your isvery.fun subdomain
- in the target field, specify your main IP or domain
- in the proxy status toggle, set the value that you specified in the form (please)

3. save the record. updating DNS takes a few minutes, but it can take longer.
